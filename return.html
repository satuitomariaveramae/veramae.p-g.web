Returns & Exchanges

Need to return or exchange a product? Follow these simple steps:

  1. Go to your Order History.
  2. Select the item you want to return or exchange.
  3. Click on “Return” or “Exchange” and fill out the form.
  4. Print the return label and send the item back.

Returns must be made within 30 days of purchase. Items must be unused and in original packaging.
